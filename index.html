Welcome to FinBot

Your advanced Discord bot for server management and custom functions. FinBot is designed to provide seamless server control, allowing users to program custom functions for tasks like displaying match results, moderating channels, and more. With its powerful features, FinBot outperforms popular bots like StartIT, Carl-bot, and Mee6, offering unparalleled flexibility and performance for Discord communities of all sizes.

API

FinBot is built using Python for core logic, Rust for performance-critical parts, and PyQT for the UI. Bot control via console.
